--- category: perl layout: post tags: - TT2 - perl title: print raw TT2 syntax --- {% include JB/setup %} The tricky is TAGS: http://search.cpan.org/perldoc?Template#TAGS

[% a = 1 %]

var [% a %] blabla;

[% TAGS <$ $> %]
var [% a %] blabla

<$ TAGS [% %] $>
var [% a %] blabla;

it will print out stuff like:

var 1 blabla;
var [% a %] blabla
var 1 blabla;

simple, set tags to <$ $> and get it back to [% %]. <$ $> can be anything you like.
