--- category: linux layout: post tags: - Apache title: 'Apache Tip: AddType' --- {% include JB/setup %} see I setup a http://code.fayland.org/ for code review. then I want to show plain text for pl file like Makefile.PL
but by default, it's handled by perl. and it shows a 500 error in browser.

with the dreamhost support, I create a .htaccess like

RemoveHandler .pl .PL .t .html
AddType text/plain .pl .PL .t .html

and it works!
